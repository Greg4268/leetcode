# 9. Palindrome Number

**Difficulty**: Easy
**Topics**: Math 
**LeetCode Link**: https://leetcode.com/problems/problem-name/\

## Problem Statement

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

**Example:**
```
Input: x = 121
Output: true
Explanation: 121 reads as 121 from left to right and from right to left 
```

**Constraints:**
- -1^31 ≤ x ≤ 2^31-1
